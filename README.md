# 🍓 My Raspberry Pi Lab

## 1. About This Project
This repository serves as a bridge between **software theory** and **hardware control**, inspired by the Harvard **CS50x** curriculum. It documents my journey of exploring computing principles through a hands-on approach with Raspberry Pi.

## 2. Hardware Specifications
*   **Main Board:** Raspberry Pi 4 Model B (4GB RAM)
*   **Peripherals:** 16GB MicroSD Card, 5V 3A Power Adapter

## 3. Software Environment
*   **OS:** Raspberry Pi OS (64-bit)
*   **Languages:** C (GCC), Python 3.x
*   **Tools:** VS Code, Terminal (SSH)

## 4. Repository Structure
*   **/docs**: Installation procedures and hardware configuration logs.
*   **/scripts**: Shell scripts for system automation and environment setup.
*   **/exercises**: Basic hardware control practice using C and Python.
*   **/projects**: Independent small-scale projects.

## 5. Goals & Features
*   Understand **Low-Level computing** and OS principles.
*   Gain experience in **direct hardware control** using the C language.
*   Build a foundation for **Edge AI** and **System Security**.

## 6. Getting Started
1.  Clone this repository:
    ```bash
    git clone [https://github.com/your-username/Raspberry-Pi-Lab.git](https://github.com/your-username/Raspberry-Pi-Lab.git)
    ```
